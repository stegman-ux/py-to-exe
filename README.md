> **With this Python tool, you can use pyinstaller , to convert .py file in .exe**

- **`Windows` Installation**

```text
- Download the .zip
- Extract it
```

- **`Linux` Installation**

```shell
git clone https://github.com/intrable-fr/py-to-exe/
```

> **Execution**

```shell
python3 py-to-exe.py
```
